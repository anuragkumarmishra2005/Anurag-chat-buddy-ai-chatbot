# Anurag AI Chatbot 🤖

An AI-powered chatbot web application built using React and Vite.
This project provides a modern chat interface where users can send messages and receive AI responses.

## 🚀 Features

* Modern chat UI
* Real-time message interaction
* Built with React components
* Fast development server using Vite
* Responsive design

## 🛠️ Technologies Used

* React
* Vite
* JavaScript / TypeScript
* Node.js
* CSS / Tailwind (if used)

## 📦 Installation

1. Clone the repository
   git clone https://github.com/your-username/anurag-ai-chatbot.git

2. Go into the project folder
   cd anurag-ai-chatbot

3. Install dependencies
   npm install

4. Start the development server
   npm run dev

5. Open in your browser
   http://localhost:8080
